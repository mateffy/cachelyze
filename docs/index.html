cachelyze

Stop re-exploring. Start caching.

Every agent session wastes tokens re-reading your codebase. cachelyze runs a full analysis once per commit, caches it, and serves it instantly — then runs a fast diff pass for any uncommitted changes.

Get codebase context in your agent in one command

Installation & Quickstart

$npm install -g cachelyze
$cachelyze --harness claude
$cachelyze --prompt "explain the auth system"
Read the full documentation →

Features

One analysis per commit,
served instantly

The first run on a commit invokes your configured AI harness to explore the codebase and write a dense analysis. Every subsequent call — in any process, session, or agent — returns it from cache instantly. Switch commits and the right cache entry loads automatically.

Organic diff analysis

Working changes are never passed as a raw diff. The agent discovers them organically via its own git tools, producing richer insight than a plain patch.

Pluggable runner

Implement HarnessRunner to use any agent — subprocess, in-process library, remote API, or test mock. Supports opencode, claude, codex, aider, gemini out of the box.

Pluggable cache store

FsStore for the filesystem, MemoryStore for in-process use, or bring your own — implement CacheStore to persist analyses in a database, Redis, or S3.

Full TypeScript SDK

Everything is exported. Use analyze() directly in your application, wire up custom stores and runners, handle errors with typed exceptions. Zero magic, full control.

Integrate into your application using the TypeScript SDK

example.ts 
import { analyze, FsStore } from "cachelyze"
import type { HarnessRunner, RunRequest, RunResult } from "cachelyze"

// Custom in-process runner — no subprocess needed
class MyAgentRunner implements HarnessRunner {
  readonly name = "my-agent"
  async run(req: RunRequest): Promise<RunResult> {
    const output = await myAgent.query(req.prompt, { cwd: req.cwd })
    return { output }
  }
}

// Persistent cache + custom runner
const result = await analyze({
  runner: new MyAgentRunner(),
  store:  new FsStore(),
  prompt: "what changed in the auth layer?",
  maxAge: 2 * 60 * 60 * 1000, // 2 hours
})

console.log(result.analysis)
console.log(result.fromCache) // true if base was cached
